1>A cosa serve Bootstrap?

Bootstrap è un framework front-end per uno sviluppo web più semplice, veloce e reattivo

2>Cosa devi importare per creare una pagina web con bootstrap

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.1/dist/css/bootstrap.min.css" rel="stylesheet">

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.2.1/dist/js/bootstrap.bundle.min.js"> &inscript>

3>Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

importare bootstrap al posto di scaricarlo velocizza i tempi di caricamento

4>A cosa serve questo attributo del tag meta?

content="width=device-width, initial-scale=1

Per garantire un rendering e uno zoom al tocco corretti

5>Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

container-fluid fornisce un contenitore a larghezza intera ,che copre completamente la larghezza della finestra
